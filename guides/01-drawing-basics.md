# Drawing Basics

PenSculpt is an iPad drawing app designed for Apple Pencil. This guide covers the drawing features.

## Getting Started

Open the app to create a new document or open an existing `.pensculpt` file. You'll see a blank white canvas — the entire screen is your drawing surface.

## Drawing

Touch the Apple Pencil to the canvas and draw. The app captures:
- **Position** — where the stroke goes
- **Pressure** — how hard you press (affects line width)
- **Tilt** — the angle of the pencil (affects shading)

## Toolbar

The toolbar is hidden by default. To reveal it:
- Tap the **round button with three dots** (⋯) at the bottom center of the screen

The toolbar contains:
- **Brush controls** — adjust brush size and opacity (top section)
- **Undo** (arrow back) — undo the last action
- **Redo** (arrow forward) — redo an undone action
- **Drawing tools** — pen, pencil, marker, eraser, etc.
- **Clear** (trash icon) — clear the entire canvas

Tap the **chevron down** button to hide the toolbar again.

## Navigation Bar (top right)

- **Lasso / Pencil toggle** — switch between Draw mode and Select mode (for selecting strokes to sculpt)
- **Autosave toggle** — enable/disable automatic saving
- **Save button** — manually save the document

## Apple Pencil Features

- **Double-tap** (Pencil 2 / Pencil Pro): Toggles between pen and eraser
- **Hover** (Pencil Pro / M-series iPad): Shows a cursor preview

## Select Mode

1. Tap the **lasso icon** in the top right to enter Select mode
2. Draw a loop around the strokes you want to sculpt
3. Selected strokes will highlight
4. A blue **"Sculpt"** button appears at the bottom — tap it to enter Sculpt mode

## Saving

Documents save automatically when autosave is enabled. You can also save manually with the save button. Files use the `.pensculpt` format and appear in the Files app.

## Tips

- The canvas supports unlimited undo/redo
- Eraser works at the stroke level — tap a stroke to erase the whole thing
- The toolbar is extensible — more tools may be added in future updates
